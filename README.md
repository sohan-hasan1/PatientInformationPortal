# PatientInformationPortal
Project For Excel Technologies Limited


>***INSTALLATION***
```
STEP - 1
Download PatientInformationPortal From Git.

##STEP - 2
Open PatientInformationPortal on Visual Studio

##STEP - 3
Open Package Manager Console
Run the command : Update-Database
note: if occur any error please check connection string form appsettings.json

##STEP - 4
Run as <Multiple Startup Projects>

##After Run you will redirect to patient list.
## To add a new patient, users can click on the "Add Patient" 
```
>***Basic Features***
```
1) User Interface Design:
Implemented an intuitive and user-friendly interface based on the provided UI design (UI Design.pdf).
Utilized the MVC Core framework for efficient frontend development.

2) Backend Development:
Built robust backend services using Web API technology.
Implemented RESTful APIs to facilitate seamless communication between the frontend and backend.

3) Database Management:
Integrated Entity Framework Core for efficient database operations.
MS SQL Server databases.
Designed a scalable and optimized database schema including various entities like Patients Information, DiseaseInformation, and more.
Followed the repository pattern to ensure organized and maintainable data access.
```
